Book Store

Overview
A simple web application built with ASP.NET Core and Blazor to manage a list of books. It allows users to view, add, edit, and delete books with a basic navigation menu.

Features
Display a list of books (title, author, price).
Add, edit, and delete books.
Navigation between Home and Books pages.
Basic notification system.

Tech Stack
Framework: ASP.NET Core
UI: Blazor (Server-side)
Styling: Bootstrap
Language: C#

Getting Started
Clone the repository.
Open in Visual Studio.
Restore NuGet packages and build the solution.
Run the project (default port: localhost:7134).
Navigate to /books to see the book list.
